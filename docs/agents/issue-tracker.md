# Issue Tracker

Issues live in **Linear**, workspace `emrcel`, team `MAR`.

URL: https://linear.app/emrcel/team/MAR/active

## How to create and manage issues

Use the **Linear MCP server** available in Claude Code sessions. Do not use `gh` CLI or local markdown files.

Key MCP tools:
- Create/update issue: `mcp__plugin_linear_linear__save_issue` (team: MAR)
- List issues: `mcp__plugin_linear_linear__list_issues`
- Get issue: `mcp__plugin_linear_linear__get_issue`
- Comment: `mcp__plugin_linear_linear__save_comment`
